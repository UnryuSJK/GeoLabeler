

# 项目名称：GeoLabeler-Cloud (云原生遥感样本标注工具)

## 1. 项目定位
一款基于 Python 的轻量化桌面应用，支持加载本地 `Shapefile` 样本点，并通过 API 实时调用云端（如 Sentinel-2, Landsat）影像数据，生成单点时间序列曲线，辅助用户快速完成地物分类标注。

---

## 2. 核心功能需求

### A. 矢量数据管理 (Vector Management)
* **读取功能：** 支持读取本地 `.shp` 格式的点位数据，自动识别坐标系（CRS）。
* **属性表同步：** 界面实时显示选中点的属性字段（如 ID、原始类别、备注等）。
* **标注导出：** 标注结果实时存入内存，支持一键覆盖原文件或导出为新的 `.shp/GeoPackage`，保留所有地理信息。

### B. 在线影像与底图模块 (Online Data & Basemap)
* **交互式底图：** 接入高分辨率在线地图（Google Satellite/Bing/ESRI）作为定位背景。
* **云端数据检索：** * 用户可配置参数：数据源（Sentinel-2 或 Landsat）、波段组合、**起止日期**（如 2024-01-01 至 2024-12-31）。
    * 通过 **STAC API** 或 **Google Earth Engine API** 远程检索指定坐标范围内的影像元数据。

### C. 实时时序分析模块 (Real-time Time-Series) — **核心特色**
* **异步点采样 (Async Sampling)：** 当用户在地图上点击某个点时，程序在后台线程向云端发起请求，提取该点在时间轴上的所有有效像素值（如 B4, B8 波段）。
* **自动去云处理：** 调用云端质量波段（QA Band），自动剔除受云干扰的观测点，确保曲线平滑可靠。
* **动态曲线面板：** * **指标计算：** 实时计算并绘制 NDVI、EVI、NDWI 等植被/水体指数折线图。
    * **多指标对比：** 支持在同一图表中切换或叠加显示不同指数。
* **时空联动：** 点击折线图上的某个时间节点，地图窗口自动请求并加载该日期对应的**单时相遥感瓦片**，供用户目视比对。

### D. 高效标注交互 (Fast Labeling)
* **快捷键标注：** 用户通过数字键（0-9）快速为当前点赋予分类标签，程序自动保存并逻辑跳转到下一个“待标注”点。
* **标注进度条：** 实时显示“已标/总数”比例。
* **视觉反馈：** 已标注的点在地图上改变颜色（例如从红色变为绿色），并高亮当前正在编辑的点。

---

## 3. 技术栈建议

* **GUI 框架：** `PyQt5` 或 `PySide6`（用于构建复杂的交互界面）。
* **地理数据处理：** `geopandas` (矢量), `shapely` (空间逻辑)。
* **云端连接：** * 方案 A：`pystac-client` + `stackstac` (对接 Microsoft Planetary Computer，免费且响应快)。
    * 方案 B：`ee` (Google Earth Engine Python API，功能最强，但需科学上网环境)。
* **数据可视化：** `matplotlib` 或 `pyqtgraph` (嵌入 GUI 的动态折线图)。
* **异步网络：** `requests` 或 `httpx` 结合 `QThread`，防止网络请求导致界面卡死。

---

## 4. 业务流程示例

1.  **加载：** 用户打开程序，导入一份名为 `sample_points.shp` 的文件，设置影像源为 `Sentinel-2`，时间跨度为 `2025年全年度`。
2.  **触发：** 点击地图上的 001 号点。
3.  **获取：** 程序后台调用 STAC API，获取该点 2025 年共 70 余次观测的 NDVI 值。
4.  **研判：** 界面下方弹出折线图，显示该点在 7-8 月绿度激增，10 月迅速回落。
5.  **标注：** 用户通过曲线特征判断其为“单季稻”，按下数字键 `3`，程序自动跳至 002 号点。
6.  **完成：** 所有点标注完成后，点击保存，退出。
